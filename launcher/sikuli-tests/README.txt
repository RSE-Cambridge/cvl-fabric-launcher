To run these tests:

1. Download Sikuli Windows 32bit version from https://launchpad.net/sikuli/+download

   Unpack anywhere.

2. Install the Java JDK.

3. Run a test, for example:

   C:\>Sikuli-IDE-1.0.0-Win32\sikuli-script.cmd -r c:\cvl-fabric-launcher\sikuli-tests\launcher01.sikuli

4. To use the IDE:

   C:\>Sikuli-IDE-1.0.0-Win32\sikuli-ide.cmd
