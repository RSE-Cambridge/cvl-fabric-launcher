
I generate the MASSIVE.ico Windows icon on a Windows XP virtual machine, running in VMWare Fusion.

I use the png2ico utility, which can be found here: http://www.winterdrache.de/freeware/png2ico/

Z:\Desktop\cvl_svn\launcher\trunk>png2ico MASSIVE.ico MASSIVElogoTransparent128x128.png MASSIVElogoTransparent64x64.png MASSIVElogoTransparent48x48.png MASSIVElogoTransparent32x32.png MASSIVElogoTransparent16x16.png
