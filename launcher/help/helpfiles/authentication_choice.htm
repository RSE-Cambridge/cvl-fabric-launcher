Using an SSH Key Pair or a Password

SSH Key Pair
A passphrase-protected private key file will remain on this computer after the Launcher exits.
Password
Your password will be required every time you log in. A temporary key-pair will be created, but it will be deleted when you close the remote desktop window.

If this computer is shared by an number of people, then passwords are preferable.

If this computer is not shared, then an SSH key pair will give you advanced features for managing your access.

You can switch between "SSH Key mode" and "Password mode" from the "Identity" menu.
