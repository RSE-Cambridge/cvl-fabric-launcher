New Authentication Method

This new version of the Launcher uses an authentication technique known as SSH Agent Forwarding.

When you start a session for the first time, the Launcher will generate a local passphrase-protected key on your computer which is automatically loaded into memory and "forwarded" to any remote servers which need to confirm your identity. You will be asked to enter a passphrase. After you've done this, your passphrase will be the primary method of authentication for the Launcher.

When used with the CVL Desktop, this new authentication method allows you to create file system mounts to remote HPC systems, and in the future it will support launching remote HPC jobs. To set up access to a HPC file system please use the "Remote File Manager" application by clicking on the keys icon on the CVL Desktop's task bar panel.
